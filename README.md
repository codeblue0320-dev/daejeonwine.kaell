# 🍷 자양동 오늘의 와인 (설치형 웹앱 · PWA)

위치 날씨에 맞춰 매일 1~3만원대 와인 1종을 추천하는 앱입니다.
**GitHub에 파일만 올리면 끝**이고, 폰에서 "홈 화면에 추가"하면 아이콘이 생기고 전체화면으로 **진짜 앱처럼** 실행됩니다. (Expo·Node·터미널 전부 필요 없음)

## 이 앱의 특징

- **현재 위치 날씨 자동** — 열면 내 위치의 오늘 날씨를 실시간으로 가져옵니다. (위치 거부 시 직접 입력 가능)
- **날씨 기반 추천** — 더울수록 산미·과실·시트러스·화이트, 비·저온일수록 바디·타닌·레드.
- **한 달 중복 없음** — 최근 30일 추천은 자동 제외 (폰에 저장).
- **설치형(PWA)** — 홈 화면 아이콘, 전체화면 실행, 오프라인에서도 열림.
- **날씨 실험실** — 기온·강수 슬라이더로 다른 조건 추천 미리 보기.

## 파일 목록 (전부 같은 폴더에)

```
index.html            앱 화면
app.js                날씨 호출 + 추천 알고리즘 + 30일 중복 방지
wines.js              와인 데이터 283종
manifest.json         앱 정보(이름·아이콘·색)
service-worker.js     오프라인 캐싱
icon-192.png / icon-512.png / icon-maskable-512.png / apple-touch-icon.png   아이콘
```

## GitHub Pages에 올리기 (클릭만으로)

1. [github.com](https://github.com) 로그인 → 우측 상단 **+ → New repository**.
2. 이름 입력(예: `wine`) → **Public** → **Create repository**.
3. 새 저장소 화면에서 **uploading an existing file** 클릭.
4. 위 **파일 전부(아이콘 4개 포함)** 를 드래그해서 올리고 **Commit changes**.
5. 상단 **Settings → Pages**.
6. **Source** = `Deploy from a branch`, **Branch** = `main` / `/(root)` → **Save**.
7. 1~2분 뒤 주소가 뜹니다: `https://<내아이디>.github.io/wine/`

> ⚠️ 중요: PWA(설치·오프라인)는 **https 주소에서만** 동작합니다. GitHub Pages 주소(https)는 조건을 만족하니 그대로 쓰면 됩니다.

## 폰에서 앱으로 설치하기

- **아이폰(Safari)**: 위 주소 접속 → 하단 **공유(⬆️)** → **홈 화면에 추가**.
- **안드로이드(Chrome)**: 접속하면 상단에 **"설치"** 배너가 뜹니다 → 탭. (안 뜨면 우측 ⋮ → "앱 설치")

설치하면 홈 화면 와인잔 아이콘으로 바로 열 수 있어요.

## 알림에 대해

이 버전은 앱을 **열었을 때** 그날 날씨로 추천합니다.
"앱을 닫아둬도 매일 오전 9시에 자동으로 오는 푸시 알림"은 웹앱만으로는 어렵고, 별도의 서버 또는 앞서 만든 Expo 앱이 필요합니다. 원하시면 그 부분을 따로 붙여 드릴 수 있어요.

## 바꾸고 싶을 때

- **기본 위치**: `app.js` 상단 `LAT`, `LON`.
- **중복 방지 기간**: `app.js` 의 `recentIds(hist, 30)` 의 `30`(일).
- **와인 데이터**: `wines.js` (`ok:false` 는 추천 제외).
