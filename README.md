# 🍷 자양동 오늘의 와인

대전 동구 자양동 날씨에 맞춰 매일 와인 1병을 추천하는 싱글 페이지 앱.

## 파일 구성

| 파일 | 설명 |
|---|---|
| `index.html` | 앱 전체 (HTML/CSS/JS + 데이터 임베드, 의존성 없음 — 브라우저로 열면 바로 동작) |
| `recommend.js` | 매일 아침 예약 작업용 CLI 스크립트 — `node recommend.js <최고기온> <강수mm>` |
| `wines.json` | 와인 200병 데이터 (이마트 대전복합터미널점 전체 매대, 2026.07 판독) |
| `history.json` | CLI 추천 기록 (30일 중복 방지용, 자동 갱신) |

## 와인 데이터 (`wines.json`)

한 병당 필드:

```
id      고유 번호 (1~200)
t       가격대: 1만 | 3만 | 5만 | 10만
vno,v   품종 번호(1~69) · 품종/스타일명
c       red | white | sparkling | dessert | rose
n, kn   영문명 · 한글명
r       산지 · 국가
price   판매가(원)
ok      재고 여부 (false = 앱 구매 버튼 비활성, 결품 추정)
second  세컨드 픽 여부
cep     세파주(품종 구성)
fl, pr  맛·향(% 포함) · 마리아주
sw,ac,tn,bd,ab  당도·산도·타닌·바디(0~100) · 도수(%)
desc    병별 설명
```

## 추천 알고리즘

- 데일리 풀: **1~3만원대 · 재고 · 디저트 제외** (83병). 실험실에서는 전 가격대·전 타입 탐색 가능.
- 기온↑ → 목표 산도↑, 화이트·스파클링·시트러스 가중 / 강수·저온 → 목표 바디↑, 레드·타닌 가중.
- 점수 상위 10병 중 날짜 시드 난수로 1병 선택 → **같은 날 몇 번을 돌려도 동일 결과**, 최근 30일 추천은 자동 제외.

## 예약 작업 (매일 아침)

1. 자양동(대전 동구) 날씨 검색 → 최고기온(°C)·강수량(mm) 결정
2. `node recommend.js <기온> <강수>` 실행 → 오늘의 와인 JSON 출력
3. `index.html`의 `EMBEDDED_WEATHER` 상수 한 줄을 실측값으로 교체하면 앱 날씨도 갱신됨

데이터 출처: 이마트 대전복합터미널점 매대 영상 판독(2026.07.10) — 가격·재고는 촬영 시점 기준.
