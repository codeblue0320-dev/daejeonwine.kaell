# 🍷 오늘의 와인 & 🍇 포도 품종 (설치형 웹앱 · PWA)

GitHub에 파일만 올리면 되는, 설치형 웹앱입니다. 폰에서 "홈 화면에 추가"하면 앱처럼 실행됩니다. (Node·터미널 불필요)

## 세 개의 탭

1. **🍷 와인 추천** — 위치 날씨에 맞춰 매일 1~3만원대 와인 1종 추천 (한 달 내 중복 없음, 날씨 실험실 포함).
2. **🍇 오늘의 품종** — 매일 포도 품종 하나를 자동 선정해 학습. 원어명·한글명, 맛·향, 고향, 유전적 유래, 주요 산지, 떼루아, 식별 단서, 마리아주와 **위키피디아 사진**을 보여줍니다. 150품종을 한 바퀴 돌 때까지 겹치지 않습니다.
3. **📝 퀴즈** — 사진과 특징만 보고 **원어명을 서술식으로 입력**해 맞히는 퀴즈. 금요일에는 그 주에 학습한 품종으로 복습 퀴즈가 뜨고, 평소엔 전체 품종으로 연습할 수 있습니다.

## 알림 (매일 12시 / 금요일 퀴즈)

- 알림은 "🍇 오늘의 품종" 탭의 **🔔 알림 켜기** 버튼으로 켭니다.
- **켜는 방식**: 앱을 열면 12시 이후 그날 품종 알림이 뜨고, **설치된 안드로이드(Chrome)** 에서는 앱을 닫아둬도 백그라운드로 하루 한 번 알림이 옵니다.
- 아이폰(iOS)은 브라우저 정책상 백그라운드 자동 알림이 제한적이라, 홈 화면에 설치한 뒤 앱을 열었을 때 그날 품종/퀴즈 알림이 표시됩니다.
- "앱을 완전히 꺼놔도 iOS까지 정확히 매일 12시 푸시"가 꼭 필요하면 별도 서버(웹푸시)가 필요합니다 — 원하시면 붙여 드릴 수 있어요.

## 파일 목록 (전부 같은 폴더에)

```
index.html            앱 화면(탭 3개)
app.js                로직(추천·품종·퀴즈·알림)
wines.js              와인 데이터 283종
varieties.js          포도 품종 데이터 150종(상세 프로필)
manifest.json         앱 정보
service-worker.js     오프라인 캐싱 + 백그라운드 알림
icon-192.png / icon-512.png / icon-maskable-512.png / apple-touch-icon.png
```

## GitHub Pages에 올리기 (클릭만으로)

1. [github.com](https://github.com) → **+ → New repository** → 이름(예: `wine`) → **Public** → **Create**.
2. **uploading an existing file** 클릭 → 위 **파일 전부(아이콘 포함)** 드래그 → **Commit changes**.
3. **Settings → Pages** → Source `Deploy from a branch`, Branch `main` `/(root)` → **Save**.
4. 1~2분 뒤 주소: `https://<내아이디>.github.io/wine/`
5. 폰에서 접속 → **홈 화면에 추가**(iOS: 공유 → 홈 화면에 추가 / 안드로이드: "설치" 배너).

> PWA(설치·오프라인·알림)는 **https 주소에서만** 동작합니다. GitHub Pages(https)는 조건 충족.

## 커스터마이즈

- **알림 시각**: `app.js`의 `maybeNotify`에서 `now.getHours() < 12`의 `12`를 바꾸세요.
- **위치**: `app.js` 상단 `LAT`, `LON`.
- **품종 데이터**: `varieties.js` (원어명·한글·고향·유전·떼루아·산지·식별단서·wiki).

## 데이터 참고

포도 품종 상세(고향·유전적 유래·떼루아·식별 단서)는 와인 전문 지식을 바탕으로 작성했으며, 유전적 유래가 학계에서 확정되지 않은 품종은 "밝혀지지 않음/추정"으로 표기했습니다. 사진은 위키피디아 문서의 대표 이미지를 실시간으로 불러오며, 일부 품종은 이미지가 없을 수 있습니다.
